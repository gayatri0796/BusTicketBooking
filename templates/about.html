About the Developer

This Bus Ticket Booking system was built using Flask (a Python web framework) and MySQL as the backend database. The frontend is powered by Bootstrap 5 for responsive design and styling. The site uses Jinja2 templating to render dynamic content like categories and ticket data. This project was created to demonstrate full-stack development skills with real-time integration between the user interface and database operations.
